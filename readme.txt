== Description ==

Genera confianza en tu tienda online y aumenta las ventas con weecomments. http://weecomments.com Muestra un widget de opiniones de la tienda online, se envian emails de invitación a dejar una opinión sobre los últimos pedidos. Solo opiniones de clientes reales. Rich Snippets de google, mejora el SEO. Valoraciones por producto.


== Installation ==

Para empezar a funcionar instala en tu sitio la extensión de weeComments y configúrala con la API key que encontrarás en tu panel de control de weeComments.com.


== Frequently Asked Questions ==

= ¿Cómo se deja una opinión?  =

El plugin enviará un correo con un enlace a WeeComments que le permitirá valorar distintos aspectos sobre cada pedido. 

= ¿Es un servicio gratuito? =

Sí, es gratuito y para siempre.

= ¿Qué ventajas tienen los planes PRO y PREMIUM? =

Con los planes PRO y PREMIUM se obtienen una serie de ventajas:

- Mayor número de invitaciones
- Rich Snippets
- Importación de opiniones de otro sitio
- SEO plus
- Publicaciones automáticas en Facebook
- Y mucho más
